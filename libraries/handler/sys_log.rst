-------------

  public function __construct (
      array $config = []
  )

  +-------+-------------+
  |名称   |说明         |

  |config |支持参数     |
  +-------+-------------+
  
  


__destruct()
------------
析构函数

.. code-block:: php

  public function __destruct ()



::

    关闭日志连接


write()
-------
写入日志

.. code-block:: php

  public function write (
      string $str,
      string $type = "INF",
      array $config = []
  ) : bool


:参数:
  +-------+-------------------------------+
  |名称   |说明                           |
  +=======+===============================+
  |str    |要写入的日志主体内容           |
  +-------+-------------------------------+
  |type   |日志类型，                     |
  +-------+-------------------------------+
  |config |传入的其他参数                 |
  +-------+-------------------------------+
  
  


::

    参数 `$config` :
    支持参数[priority]
