write()
-------
写入日志

.. code-block:: php

  abstract public function write (
      string $str,
      string $type = "INF",
      array $config = []
  ) : bool


:参数:
  +-------+-------------------------------+
  |名称   |说明                           |
  +=======+===============================+
  |str    |要写入的日志主体内容           |
  +-------+-------------------------------+
  |type   |日志类型，                     |
  +-------+-------------------------------+
  |config |传入的其他参数                 |
  +-------+-------------------------------+
