====
File
====

文件处理器配置
================

+---------------+-----------------------------------------------------------+---------+---------------------+
|参数名         |说明                                                       |是否可选 |默认值               |
+===============+===========================================================+=========+=====================+
|path           |日志文件保存目录                                           |是       |'./data/log'         |
+---------------+-----------------------------------------------------------+---------+---------------------+
|file           |日志文件名                                                 |是       |date('Ymd') . '.log' |
+---------------+-----------------------------------------------------------+---------+---------------------+
|max_size       |日志文件大小，超过该值将新建卷                             |是       |2 * 1024 * 1024      |
+---------------+-----------------------------------------------------------+---------+---------------------+
